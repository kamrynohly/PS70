My Microcontroller Project

For the microcontroller project, I decided to see if I could get the ESP32 XIAO to play a few tones on a speaker, resembling Sabrina Carpenter's song espresso!

I started this project with an Arduino template for playing melodies, by Tom Igoe in the ESP32 library. I thought this would be relatively straightforward, until I discovered that the XIAO does not have DAC pins, which would allow for me to communicate necessary info to the speaker. I then attempted to find a simple workaround for the XIAO before deciding to pivot to an ESP32-S3 Dev Board, which could support functionalities such as the built-in "tone" function. I used pin 8 to connect as an output, and I proceeded to test different tones at different hertz in order to replay the melody!

This was so much fun, and much more friendly to do once I had the proper set-up! Here is my code, as well as a video of the song! 

    
/*
    EspressoSong
    
    Plays a line of Espresso by Sabrina Carpenter!
*/ 
#include "pitches.h"
      
// Verse - "I can't relate to desperation, my give a **** are on vacation"
int verse[] = {
    NOTE_E4, NOTE_E4, NOTE_C4, NOTE_G4, NOTE_G4, NOTE_A4, NOTE_C5, NOTE_D5, NOTE_C5,
    NOTE_F4, NOTE_E4, NOTE_C4, NOTE_G4, NOTE_G4, NOTE_A4, NOTE_C5, NOTE_D5, NOTE_E5
};

// Each value corresponds to a 1/x note, where 1/8 is an eighth note, 1/4 is a quarter note, etc.
int verseNoteDurations[] = {
    4, 4, 4, 1, 4, 4, 4, 4, 2,
    4, 4, 4, 1, 4, 4, 4, 2, 2
};

int melody[] = {
    // Thinkin 'bout me every night oh (pause)
    NOTE_D4, NOTE_A4, NOTE_C5, NOTE_C5, NOTE_C5, NOTE_E5, NOTE_A4, NOTE_A4, NOTE_A4, NOTE_B4, 0,
    // Isn't that sweet? I guess so
    NOTE_G4, NOTE_G4, NOTE_G4, NOTE_B4, NOTE_C5, NOTE_B4, NOTE_B4, NOTE_A4
};

int melodyNoteDurations[] = {
    8, 8, 4, 8, 8, 4, 8, 8, 4, 4, 16,
    8, 8, 4, 8, 8, 4, 8, 8
};
      
// The pin connecting power & info to the speaker
int tonePin = 8;
      
void setup() {
    // No setup needed!
}

void loop() {
    // Replay melody every 5 seconds...
    delay(5000);
    
    // Iterate over verse!
    for (int note = 0; note < 18; note++) {
        int noteDuration = 1000 / verseNoteDurations[note];
        // Start the tone to play on speaker
        tone(8, verse[note], noteDuration);
    
        int pauseBetweenNotes = noteDuration * 1.30;
        delay(pauseBetweenNotes);
    
        // Stop the tone
        noTone(8);
    }
    
    delay(2000);
      
    // Iterate over melody!
    for (int note = 0; note < 19; note++) {
        int noteDuration = 1000 / melodyNoteDurations[note];
        // Start the tone to play on speaker
        tone(8, melody[note], noteDuration);
    
        int pauseBetweenNotes = noteDuration * 1.30;
        delay(pauseBetweenNotes);
    
        // Stop the tone
        noTone(8);
    }
}
