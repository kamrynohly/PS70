My Custom Capacitive Sensor

This week, I immediately knew that I wanted to create a capacitive sensor with an RGB output to demonstrate a relation to what the sensor was detecting. I decided to create a high-five high-score sensor! The idea of the sensor is that it will measure the strength of a high-five, and if it beats the previous high-score for the best high-five, then the RGB LED will light up. My goal is to have the sensor monitor what an "ideal" high-five strength is and light up as green (for good), blue (too weak), or red (too strong). For now, it will only light up with a color if the high-five sets a new high score. Below, you can find my code attached, which is non-blocking, as well as a demo.
